# Burp 流量分析工具

解析 Burp Suite 导出的 XML 流量，自动识别漏洞测试优先级。

## 背景
Burp 抓了几百条流量，不知道先测哪个。
这个工具把最值得测的排到前面，按漏洞危害和可操作性自动评分。

## 功能
- 自动识别 IDOR、越权、价格篡改、SSRF 等漏洞点
- 三维度评分排序（性价比 / 可操作性 / 全量）
- 响应敏感数据确认（手机号、身份证、SQL报错）
- 关联请求追踪，支持导出回 Burp

## 用法
python3 analyzer_tui3.py burp导出的xml文件 目标域名关键词

## 依赖
Python 3，无需额外安装库
